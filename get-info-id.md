<summary>API GET INFO UID</summary>

## Lấy thông tin id

```http
GET https://graph.facebook.com/userID
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `access_token` | `string` | **Bắt buộc**. Token của bạn |
| `userID` | `string` | ID cần lấy thông tin |
